# 🏥 VETHACHE - Gestión Veterinaria Integral

**Vethache** es una solución web diseñada para digitalizar la operativa de clínicas veterinarias, centrada en la optimización de citas, la gestión de planes de salud y remisiones. Con el plan futuro de conseguir la centralización de historiales médicos de pacientes a falta de una sanidad veterinaria pública.

---

## 🚀 Funcionalidades Principales
* **Gestión de Citas:** Sistema optimizado para evitar cuellos de botella operativos.
* **Planes de Salud:** Control de recurrencia y rentabilidad de servicios preventivos.
* **Remisiones:** Remisiones de pacientes y casos clínicos desde otros centros.
* **Diseño Responsive:** Interfaz adaptada a dispositivos móviles (Mobile-First) para consulta en clínica o en movilidad.
* **Chatbot de Asistencia:** Asistente virtual para resolución de dudas frecuentes integrado en el frontend.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3 (Flexbox), JavaScript (ES6+).
* **Diseño:** Responsive Design con enfoque en usabilidad móvil.
* **Documentación:** Normas APA 7ª Edición.
* **Herramientas de IA:** Soporte de Gemini y ChatGPT para optimización de lógica.

## 📂 Estructura del Repositorio
* `/index.html` - Punto de entrada principal de la web.
* `/css` - Hojas de estilo y diseño responsivo.
* `/js` - Lógica del chatbot y menús interactivos.
* `/docs` - Presentación en PowerPoint y documentación del proyecto.

## 📥 Instalación y Uso
1. Clona el repositorio o descarga el archivo ZIP.
2. Abre el archivo `index.html` en cualquier navegador moderno (Chrome, Edge, Firefox).
3. Para ver la versión móvil, utiliza las herramientas de desarrollador (F12) y selecciona un dispositivo como iPhone SE.
